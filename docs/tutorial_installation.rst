Installation
============

.. contents:: :local:

This page describes the requirements and instructions for installing Spotlight.

Get Anaconda environment
~~~~~~~~~~~~~~~~~~~~~~~~

This tutorial uses an Anaconda environment to install the software used.
To download Anaconda, see: https://docs.anaconda.com/anaconda/install/

Installation
~~~~~~~~~~~~

Inside the Spotlight repository there is an Anaconda environment YAML file (``tools/environment_rietveld.yaml``) that installs Spotlight, Mystic, GSAS-II, and MILK into the same environment for this tutorial.
